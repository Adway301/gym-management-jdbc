# Gym Management System (JDBC)

## 📌 Description
This is a Java-based Gym Management System using JDBC for database connectivity.

## 🚀 Features
- Add / Remove Members
- Manage Trainers
- Database operations using JDBC

## 🛠 Tech Stack
- Java
- JDBC
- MySQL

## ▶️ How to Run
1. Open project in IntelliJ
2. Setup MySQL database
3. Run Main.java

## 📁 Project Structure
- src → source code
- lib → JDBC drivers
